2. Pega la ruta completa del archivo
3. Presiona Enter
4. El archivo se abrirá
5. Selecciona TODO (Ctrl+A)
6. Copia (Ctrl+C)
7. Pega en Supabase SQL Editor

1. Abre el Explorador de Windows
2. Pega la ruta en la barra de direcciones
3. Presiona Enter
4. Abre el archivo con tu editor de texto preferido
5. Copia todo el contenido
6. Pega en Supabase SQL Editor

