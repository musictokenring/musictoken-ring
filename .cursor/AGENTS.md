# Agentes de Cursor - Development Environment Setup D3D0

## Perfil: development-environment-setup-d3d0

Este perfil configura el entorno de desarrollo para MusicTokenRing Operativo.

### Características

- Formateo automático de código
- Auto-guardado configurado
- Linting y corrección automática
- Organización automática de imports

### Uso

Este perfil se aplica automáticamente cuando trabajas en este proyecto. Las reglas definidas en `.cursor/rules/` proporcionan contexto persistente para el asistente de IA.

### Configuración

Ver `.cursor/profile-development-environment-setup-d3d0.json` para la configuración completa del perfil.
